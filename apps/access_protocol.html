About this app

This app is created as an authentication app:
  1. Authenticates the user against Google before running other functions
  2. Retrieves files stored under the thsconline@gmail.com account for viewing
  3. Calls other scripts to return relevant HTML output

Authentication App

This script is used to authenticate users to add a layer of security before access files stored on Google Drive
Launch App

Custom Configurations

The script can be configured by adding parameters to the script url as follows

?base: the encrypted id of the folder in Google Drive
  login: Specifies the ID is not encrypted
  (numeric code): Specifies a particular filter for the query and adjusts the query input

?field: a parameter parsed to the script to run a query on

Privacy Policy & Terms of Service:

Users upon granting access to the script, grant access to any Google Drive folders & files that can be retrieved by the user. However the script does not retrieve any files.

Users email address may be displayed but are not stored remotely. No information is retained or disclosed to any other user or third-party organisation.
